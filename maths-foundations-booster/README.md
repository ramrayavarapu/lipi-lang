# Maths Foundations Booster

A standalone browser app to help children build confidence in foundational maths.

## Features
- Registration/Login with age-group profile (localStorage-based)
- Activity selection by age group
- Instant answer feedback
- Progress dashboard (attempted vs correct)

## Run
Open `index.html` in a web browser.

## Data Flow
- Registration/Login -> Authentication data fetch -> User dashboard
- Activity selection -> Activity execution -> Result and feedback
